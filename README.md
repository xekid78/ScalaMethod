# ScalaMethod
メソッド

## 処理
`sum_gokei()`メソッドを作り、その中で計算を行う。

## コード
```
object Def {
    def sum_gokei {
        println(5 * 8)
    }

    def main(args: Array[String]): Unit = {
        sum_gokei
    }
}
```

## 出力結果  
```
40
```
